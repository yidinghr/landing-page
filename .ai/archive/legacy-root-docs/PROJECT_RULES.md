# PROJECT RULES

- Đây là web nội bộ của 弈鼎.
- Toàn bộ giao diện dùng nền đen, khung vàng, chữ vàng.
- Không tự ý đổi sang phong cách trắng xanh hiện đại.
- Không tự ý đổi framework nếu chưa được yêu cầu.
- Ưu tiên mở rộng từ project hiện tại đang có trong workspace.
- Dữ liệu nhân viên sẽ lấy từ Airtable thông qua backend nội bộ.
- Mỗi lần sửa phải nói rõ:
  1. đã đọc file nào
  2. sẽ sửa file nào
  3. sẽ tạo file nào
- Chỉ tạo số lượng file tối thiểu cần thiết.
- Không được nhét toàn bộ logic vào một file index.html duy nhất.
- Mọi phần chưa chắc chắn phải kiểm tra file/project trước, không được tự bịa.
- Luôn test trên web thật rước.
